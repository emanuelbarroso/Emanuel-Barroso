GIT NO LINUX
1) Comando para clonar repositório disponível na internet: 
git clone https://github.com/emanuelbarroso/Emanuel-Barroso


2) Comando para atualizar repositório no seu computador de acordo com a versão mais recente:
(adicione aqui)


3) Comando para adicionar arquivo na lista de arquivos para commit:
git add README.txt


4) Comando para fazer o commit de uma lista de arquivos:
git commit -m 'All files were commited'


5) Comando para transferir arquivos que sofreram commit para o repositório na internet:
git push
